# Task 1: Basic Network Sniffer

## Objective
Capture and analyze network packets using Python.

## Tools Used
- Python
- Scapy

## Features
- Captures live network packets
- Displays source and destination IP
- Identifies protocols (TCP, UDP, ICMP)
- Shows port numbers
- Displays packet payload (partial)

## How to Run
1. Install scapy:
   pip install scapy

2. Run script as administrator:
   python sniffer.py

## Output Example
Shows captured packets with IP, protocol, ports, and payload.

## Note
HTTPS payload is encrypted and not human-readable.
